# Assets Directory

Place your game assets (PNG images) in this directory:

- `village-bg.png` - Background image for the village (1600x1200 recommended)
- `player.png` - Player character sprite (32x32 recommended)
- `house1.png` - House sprite 1 (80x80 recommended)
- `house2.png` - House sprite 2 (80x80 recommended)
- `house3.png` - House sprite 3 (80x80 recommended)

If these files are not present, the game will automatically generate colored placeholder rectangles.

## Asset Specifications

- **Format**: PNG with transparency support
- **Style**: Top-down 2D pixel art
- **Background**: Should be a tileable or seamless background image
- **Player**: Single frame sprite, facing down by default
- **Houses**: Static sprites, can be different styles/variations
